# Spreadsheet

Online spreadsheet

[Website](spreadsheetonline.azurewebsites.net)

## Features
1. 100 x 100 grid
2. Multiplication, division, addition, subtraction

    Ex. =4/3*3-2+1
3. Ctrl B - Bold, Ctrl I = Italics, Ctrl U - Underline
